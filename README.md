# FBA Brain v3.0 - Amazon UK Arbitrage Engine

Desktop app + iPhone app. Runs 24/7 finding profitable products.

## Setup

```
npm install
cp .env.example .env
# Add your KEEPA_API_KEY to .env
```

## Run

```
npm start          # Web server (Railway + iPhone)
npm run desktop    # Desktop app (laptop)
```

## iPhone: Open URL in Safari > Share > Add to Home Screen

## Deploy: Push to GitHub > railway.app > connect > add KEEPA_API_KEY
